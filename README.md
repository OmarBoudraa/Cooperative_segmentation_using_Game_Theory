# Region-Edge Cooperation for Image Segmentation Using Game Theory

# How to run ?

Manual :

> Install QT/C++

> Install OpenCV (ideally minimum of 3.0 version)

> Open "SCG.pro" using QT/C++ SDK

> Clean and build project
